In order to unblock a site, make the link in this format - (mmkgz.github.io/Cool?url="Type URL here")
Make sure that in order to avoid any GoGuardian blocks, replace the word 'game' with '$'. e.g "https://mmkgz.github.io/Unblock/?url=https://dedra$s.com/$s/ovo/redirect.html" for "https://mmkgz.github.io/Unblock/?url=https://dedragames.com/games/ovo/redirect.html."
